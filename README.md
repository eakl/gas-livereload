# Addon

## Install

```bash
pnpm install

# Setup the project
pnpm run preinit
pnpm run init
pnpm run postinit
pnpm run setup

# install mkcert globally
pnpm add -g mkcert

# Install https certificates
pnpm run install:https

# Build and deploy the dev wrapper
pnpm run build:dev
pnpm run clasp:push

# Start the dev server
pnpm run dev
```

Visit `https://localhost:5173/` to see the app in action.
